Installing VJET Beta 3/20/2011

 

Install VJET

 

Install Eclipse Helios

     

Eclipse IDE for Java EE Developers – Windows 32 Bit

http://www.eclipse.org/downloads/

 

Install VJET

 

1.       From within Eclipse open the update manager by selecting Help > Install New Software….

2.       Click Add… and enter the location: http://jsp/tools/vjet_beta/oss_rc

3.       Select VJET and finish the installation.

 

*If you plan to use the VJET Type Libraries, you must import the libraries into your workspace and add the libraries to your project’s build path.  Refer to the Load VJET Type Libraries section.

 

 

Load VJET Type Libraries:

 

1.       Download the VJET Type Library zip files from the following locations:

1.       Dojo - http://jsp/tools/vjet_beta/typelibs/DojoTL_3.30.2011.zip

2.       JQuery  - http://jsp/tools/vjet_beta/typelibs/JQueryTL_3.30.2011.zip

3.       NodeJS -  http://jsp/tools/vjet_beta/typelibs/NodeJsTL_3.30.2011.zip

4.       HTML 5 Canvas -  http://jsp/tools/vjet_beta/typelibs/CanvasTL_3.30.2011.zip

 

 

2.       From Eclipse open the import wizard by selecting File -> Import…

3.       In the Import wizard select General -> Existing Projects Into Workspace

4.       Select Next

5.       Select  Select archive file: , then Browse and select the project you want to import

6.       Select Finish. You will see the imported VJET Type Library project(s) in the Project Explorer. 

 

 

 

 

7.       Add Project Dependencies.   If you want to reference VJET Type Libraries (NodeJS, jQuery, Dojo, JSON) in your project, you must add them to your project’s build path.  Refer to the Getting Started section to create and add dependencies to a VJET Project.

 

 

 

 

Getting Started

 

1.       Switch to the VJET JS Perspective

·         Select Window -> Open Perspective -> Other…

·         Select VJET JS (VJET JS Browsing Perspective not yet supported)

2.        Create a VJET Project

·         Select File -> New -> VJET Project.  Enter the project name -> Next

3.       Add Project Dependencies.  If you want to reference VJET Type Libraries (NodeJS, jQuery, Dojo, JSON) in your project, you must add them to your project’s build path.

 

·         Right click on your project.  Select Build Path -> Configure Build Path

·         Select Projects tab -> Add -> select the projects you want to add.  Select -> OK -> Finish

 

4.       A newly created VJET Project is displayed below.

The project icon is annotated with a ‘V’
Default libraries include:

o   JS Native Types – JavaScript Primitives, JavaScript native types

o   Browser – DOM, HTML, Browser objects support

o   VJO LIB – VJET Types

       

 

5.       Create a JavaScript file.  VJET JavaScript authoring features apply to files with the .js file extension.

 

·         Select File -> New -> JavaScript Source File and enter a filename. 

 

Note: For now, disregard the VJET Types
