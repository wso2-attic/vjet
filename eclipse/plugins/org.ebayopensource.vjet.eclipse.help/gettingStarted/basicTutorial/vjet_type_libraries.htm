VJET JS Type Libraries

VJET JS Type Libraries provide the JavaScript metadata descriptions used to support VJET JS code assist and development features. As seen in the previous section, VJET Projects include a default set of type libraries that support basic JavaScript usage:

  • JS Native Types - JavaScript primitives and native types (eg., int, Date)
  • Browser SDK - browser and DOM types (eg., Window, HTMLElement)
  • VJO LIB - VJET JS types

Expand the folders to browse the libraries and to get a sense of the JavaScript support provided by VJET JS. These libraries support the code assist features demonstrated in the following section.
