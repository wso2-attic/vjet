Detecting existing layout

Layout on file system

  • The source files for a product are laid out in one directory "src".

Steps for defining a corresponding project

  1. Click Window > Open Perspective > Other... > VJO to change to the VJO perspective.

  2. Click File > New > Other... > VJO Project to open the New VJO Project wizard.

  3. Type "Product" in the Project name field.

  4. In Contents group, change selection to Create project from existing source.

    Click Browse... and choose the Product directory on drive D:.

  5. Click Next.

    Ensure that the source folders are detected. [Not yet implemented]

  6. Click Finish.
  7. You now have a VJO project with a "src" folder which contains the sources of the "Product" directory.
    [Not yet implemenyed]

Note: This solution creates a ".project" file and a ".buildpath" file in the "Product" directory. If you do not wish to have these files in the "Product" directory, you should use linked folders as shown in the Sibling products in a common source tree section.
