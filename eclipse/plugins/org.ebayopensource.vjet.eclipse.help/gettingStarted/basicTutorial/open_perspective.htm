Open a VJET Perspective

  • Select the menu item Window > Open Perspective > Other.... Then select VJET JS.
  • You can also click the button in the upper right corner and select Other.. to open a perspective.

The default VJET Perspective is comprised of:

  • Script Explorer - Displays VJET Projects
  • Editor area
  • Outline - Displays an outline of the current file in the editor area
  • Problems, Tasks, Console views - Informational views
