The VJET development tools contribute the following perspectives to the workbench:

  • VJET JS Perspective
  • VJET JS Browsing Perspective

VJET JS Perspective

The perspective is designed for working with VJET projects. It consists of the following views:

  • Editor Area
  • Script Explorer view
  • Outline view
  • Problems view
  • Tasks view
  • Console view

To open this perspective please through Window menu (Window > Open Perspective > Other... > VJET JS) .
