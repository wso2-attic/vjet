Templates

Templates are a structured description of coding patterns that reoccur in source code.
The VJET editor supports the use of templates to fill in commonly used source patterns.
Templates are inserted using content assist. Shortcut: Ctrl+Space.
Note: Press Ctrl+Space again, Template Proposal will be shown.
