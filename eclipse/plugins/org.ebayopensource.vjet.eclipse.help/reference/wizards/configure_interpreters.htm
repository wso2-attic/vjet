Configure Interpreters


To reach this page please through the Window menu (Window > Preferences > VJET > Interpreters) of the workbench.


Interpreters

This page is used to add, remove or edit interpreter definitions.

Interpreters Options
Option Description
Host Select an existing Host in the drop down menu.
Add Add another interpreter to this project.
Edit Edit the selected interpreter.
Copy Copy the selected interpreter.
Remove Remove the selected interpreter.
Note: The default VJET Rhino Interpreter can't be removed.
Search Search available interpreters.

Add interpreter page

This page is used to add a interpreter.
The Add, Edit, Copy options are similar, so we take Add option for example.

Add Interpreter Options
Option Description Default
Interpreter type Select the Interpreter type in the drop down list. Generic VJET Install Type
Interpreter name Input the name for the new project. <blank>
Interpreter executable Input or browse to select executable interpreter. <blank>
Interpreter arguments Input the arguments for the new interpreter. <blank>
Interpreter system libraries
Add External libraries
Add a predefined library to the interpreter system libraries.
Remove
Remove the selected element from the interpreter system libraries.
Up
Click the Up button to move the selected library entry up in the build path order for this new interpreter.
Down
Click the Down button to move the selected library entry down in the build path order for this new interpreter.
Restore Default
All changes that have been applied to the interpreter system libraries will be withdrawn and the original state is reconstructed.
