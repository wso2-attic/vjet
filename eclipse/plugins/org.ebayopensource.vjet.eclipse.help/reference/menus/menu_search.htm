Search actions

An example is shown below.

Search menu commands:
Name Function Keyboard Shortcut
Search... Opens the search dialog. Ctrl + H
File... Opens the search dialog on the File search page.  
References Finds all references to the selected VJET element.  
Declarations Finds all declarations of the selected VJET element.  
Satisfiers Finds all Itypes of the selected VJET element.  

Search Scopes Submenu:

Scope Availability Description
Workspace all elements Searches in the full workspace

Scopes can be saved and names in the working set dialog. Existing instances of working sets are also available in the Search Scope submenu.

A VJET search can also be conducted via the context menu of selected resources and elements in the following views:

  • Script Explorer
  • Outline view
  • Search result view
  • Hierarchy view
  • VJET Browsing views

The search context menu is also available in the VJET editor. The search is only performed if the currently selected text can be resolved to a VJET element.

The type of the selected VJET element defines which search context menus are available. The VJET editor does not constrain the list of available VJET searches based on the selection.

Please refer to the Search Menu Actions in Java Development User Guide for detailed introduction.
