VJET editor

Toolbar actions

Toolbar Button Command Description

Go to Next Problem This command navigates to the next problem marker in the active editor.

Go to Previous Problem This command navigates to the previous problem marker in the active editor.

Toggle Mark Occurrences Turns mark occurrences on and off in the VJET editor.

Show Selected Element Only

This button enabled display of a segmented view of the source of a source module.

This button is only shown if you customize your perspective to show the Editor Presentation actions.

For example, if a method is selected in the Outline view, the Show Selected Element Only option causes only that method to be displayed in the editor, as opposed to the entire class.

Off:
The entire source module is displayed in the editor, with the selected VJET element highlighted in the marker bar with a range indicator.
On:
Only the selected VJET element is displayed in the editor, which is linked to the selection in the Outline or Hierarchy view.

Show Whitespace Characters This button enabled display of whitespace characters in an editor. This button is only shown if youcustomize your perspective to show the Editor Presentation actions.

Please refer to the Java Editor in Java Development User Guide for detailed introduction.
