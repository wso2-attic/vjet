The VJET development tools contribute the following perspectives to the workbench:

  • VJET JS Perspective
  • VJET JS Browsing Perspective

VJET JS Browsing Perspective

The perspective is designed for browsing the structure of VJET projects. It consists of the following views:

  • Editor Area
  • Projects view
  • Packages view
  • Types view
  • Members view

The other four views are by default linked to the active editor area.

To open this perspective please through Window menu (Window > Open Perspective > Other... > VJET JS Browsing).

Please refer to the VJET views and VJET editor for detailed introduction.
